# life_tools（小蜜）

一款“把生活拆成工具箱”的 Flutter 应用：打开就是首页工具卡片，点进去就是专注的一件事。支持 Android、iOS、Web、Linux、macOS、Windows，多端数据默认全部落在本地（SQLite），不依赖云账号也能完整使用。

## 你能用它做什么

### 1) 工作记录（Work Log）

把“今天做了什么”变得可追溯、可复盘：

- 任务清单：创建/编辑任务，支持置顶、排序、筛选
- 工时记录：为任务记工时，按天/按任务回看
- 日历视图：用日历翻阅节奏与产出
- 标签体系：用公共标签统一管理（跨工具复用）
- AI 辅助：把零散描述变成结构化任务/记录（可选）
- 语音输入：语音 → 文本 →（可选）AI 整理成任务
- 操作日志：记录关键操作，方便回溯（同步/备份时会带走）

### 2) 囤货助手（Stockpile Assistant）

让“家里还有没有、什么时候会过期”这件事不用再靠记忆：

- 家庭库存：添加/编辑物品，记录数量、保质期/到期日等信息
- 消耗记录：每次消耗都能追踪，形成使用习惯数据
- 临期/过期提醒：自动生成消息并可触发系统通知（移动端）
- AI 批量录入：一段描述快速生成多条物品（可选）
- 标签筛选：按标签快速定位同类物品

### 3) 胡闹厨房（Overcooked Kitchen）

偏“轻松经营”的厨房小工具，用来管理食谱、愿望单与做饭节奏：

- 食谱：新增/编辑食谱，支持多维标签
- 愿望单：想做/想买的清单，支持提醒消息
- 计划/统计：按日记录餐食与条目，生成轻量统计

### 4) 标签管理（Tag Manager）

标签是全局能力：一个地方维护，多个工具复用。

- 维护公共标签与工具关联
- 支持分类、排序、重命名、删除
- 备份/同步时优先导入标签，避免其它工具的标签关联丢失

## 关键能力（跨工具）

- 消息中心：集中展示提醒/事件，支持过期消息清理
- 备份与还原：导出 JSON / TXT，支持分享接收并一键导入
- 数据同步：对接自建同步服务端，支持公网/私网 WiFi 模式（SSID 白名单防误同步）
- 资源存储（可选）：本地存储或七牛云；并带图片缓存机制，兼顾离线体验
- iOS 26 风格 UI：全局主题与交互规范统一

## 数据与隐私

- 默认：所有业务数据存储在本地 SQLite。
- 可选联网能力：
  - AI：需要你在“AI 设置”中自行填写 Base URL 与 API Key。
  - 资源存储：使用七牛云时需要你自行填写 AK/SK 与域名配置。
  - 数据同步：需要你自行配置同步服务端地址。
- 安全提示：为方便大多数用户跨设备迁移，“备份与还原”默认会导出敏感信息（AI Key / Token / 七牛 AKSK），你可以手动关闭该开关以避免导出；无论是否关闭，分享前都请确认接收方可信。

## 快速开始（开发）

如果 `flutter` 不在 `PATH`，可用 `~/flutter/bin/flutter`（或按 `AGENTS.md` 中的其它路径）。

```bash
# 安装依赖
flutter pub get

# 运行应用
flutter run

# 静态分析 + 测试
flutter analyze
flutter test

# 格式化
dart format .
```

## 目录结构（概览）

- `lib/main.dart`：应用入口 + Provider 注入（AiService / ObjStoreService / TagService / MessageService 等公共入口）
- `lib/core/**`：通用能力（AI、同步、备份、消息、通知、对象存储、主题、数据库等）
- `lib/tools/**`：各个“工具”的业务实现
- `test/**`：单元/组件/页面测试（建议新增功能先补测试）
